# Power in Numbers - Vue 3 Prototype

A collaborative project budgeting tool that bridges individual financial needs with collective resource allocation.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- **Financial Profiles**: Calculate Goal, Now, and Minimum rates based on real expenses
- **Budget Scenarios**: Create multiple "what-if" budgets (Dream, Realistic, Bare Bones)
- **Equity Tracking**: Track "Producerial Shares" when collaborators accept lower wages
- **Project Management**: Organize projects into phases with income source tracking

## Demo Data

The app comes pre-loaded with demo data including collaborators, projects, and budget scenarios. All data persists in browser localStorage.

## Tech Stack

- Vue 3 (Composition API)
- Pinia (State Management)
- Vue Router
- Vite
- Vanilla CSS (Modern Design System)
